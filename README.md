# Financas

> ## Descrição
>
> - Aplicativo para controle de finanças pessoais.
> 
<br/>

> ## Status
> Em desenvolvimento
<br/>

> ## Tecnologias
>
> - linguagem: Kotlin
> - padrão de arquitetura: MVVM + Clean architecture
> - chamadas assíncronas: Coroutines
> - lib de persistência: Room
> - injeção de dependiência com Koin
>
